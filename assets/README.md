# Phase-Fiction Assets

The current SVG assets are retained as compatibility placeholders, but their intended meaning in this repository is now fiction-first rather than their original workflow branding:

- The outer brackets represent the active story-phase boundary.
- The three stacked cards represent the fixed context window: story constraints, current phase, and execution boundary.
- The flowing line represents `planctl` advancing through premise, cast, plotting, drafting, and revision phases one step at a time.
- The terminal node represents objective close-out: `complete` and `finalize` writing the current manuscript state back to `state.yaml` and `handoff.md`.

Files:

- `phase-contract-logo.svg` - current horizontal lockup asset, kept under its legacy filename for link compatibility.
- `phase-contract-mark.svg` - current standalone mark asset, also retained under its legacy filename.
