# Part 3 Matrix

## 终局场景矩阵

1. 信号盲区里出现真假难辨的“安全转移”命令，围猎正式开始。
2. 林雁与韩松原被迫短暂结盟，抢在吴凯霖完全锁死系统前转移阿茹娜。
3. 乔静雪阵营在终点前缓行区发动最后一次假交接，企图合法接走证人。
4. 林雁借广播、名单与调度单把责任链从私人凶案拉到公共秩序层面。
5. 终点站总摊牌：直接动手责任、系统维稳责任和企业利益责任同时暴露。
6. 真相落地后的余波：证词保住了，但所有人都必须承担活下来的代价。

## 收束功能

- 场景 1：制造最后一次信息混乱和空间追逐。
- 场景 2：把临时联盟推到最危险的位置。
- 场景 3：让“正式程序”本身变成杀人工具。
- 场景 4：把私人发现变成公开证据。
- 场景 5：完成终点站的责任拆解与情感爆裂。
- 场景 6：留下现实代价和后续余震。
