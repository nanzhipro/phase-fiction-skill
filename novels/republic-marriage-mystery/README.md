# 《回门之前》

《回门之前》是一部民国宅门悬疑长篇。故事从大户人家大小姐裴见月下嫁寒门书生陆子衡开始：所有人都以为这是一场任性婚事，只有她知道，自己是带着一页被人撕开的旧账本和一封不敢拆开的遗信嫁出去的。

婚后第一个夜里，陪嫁丫鬟失踪，裴家家主暴死，寒柳巷旧宅里出现本该被烧毁多年的嫁妆花押。七日后是回门宴，也是裴家族产重分与旧案证据彻底消失的最后期限。裴见月必须在新婚身份、阶层目光和夫妻互疑之间找出真相，否则她既回不了娘家，也活不到回门那天。
